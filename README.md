## 📖 Project Overview

**Neon Tic-Tac-Toe** is a web-based, interactive logic game built entirely from scratch without any heavy frameworks. It serves as an exploration into advanced DOM manipulation and modern CSS rendering techniques. 

The core Vanilla JavaScript logic is wrapped in an ultra-modern, glassmorphism UI experiment, featuring vibrant neon glows, smooth state transitions, and responsive layout scaling.

---

## ✨ Features

- **Zero Dependencies**: Pure, unadulterated HTML, CSS, and Vanilla JavaScript. No React, no Vue, no bloated libraries.
- **Glassmorphism UI**: Beautiful, frosted glass panels with backdrop blurring and dynamic CSS variables.
- **Neon Aesthetics**: Cyberpunk-inspired neon glowing animations on interactions and win-states.
- **Responsive Logic**: Flawless game state management, win-detection algorithms, and automated resets.

## 🛠 Tech Stack

| Layer       | Technology                                          |

| **Style**   | CSS3, CSS Variables, Flexbox/Grid, Animations       |
| **Structure**| Semantic HTML5                                     |

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ishanbhattacharjee12/Tic-Tac-Toe-Game-Project.git
   cd Tic-Tac-Toe-Game-Project
   ```

2. No installation required! Simply open the `index.html` file in any modern web browser to play.
   ```bash
   open index.html
   ```
